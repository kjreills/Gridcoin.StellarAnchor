Gridcoin Stellar Anchor

Purpose

This project aims to make Gridcoin more visible and easier to purchase by making it available on the Stellar decentralized exchange. Specifically, this anchor makes it possible to exchange real Gridcoin (GRC) for a tokenized version of it on the Stellar blockchain.

The Stellar DEX has very low transaction fees and some cool projects being built on it. Check out Lumenswap for a quick and easy way to swap any two assets on the Stellar DEX!
